# helika-unity-sdk
Helika Unity SDK

## To open the Project
In the terminal, run
`<path-to>\Unity.exe -gvh_disable -projectPath <path-to-folder> -exportPackage Assets\Helika Helika.unitypackage`

## Build Unity Package
In the termminal, run:
`<path-to>\Unity.exe -gvh_disable -projectPath <path-to-folder> -exportPackage Assets\Helika Helika.unitypackage`
